Typical slide structure

                    
<div class="step" id="ID_SLIDE" data-scale="1" data-rotate="0" data-x="0" data-y="0" data-z="0">
    <div class="slide-border">
        <div class="slide-body">
            <h1>SLIDE_TITLE</h1>
            <hr>
            Class 'slide-body' is used for what is visible only in this slide
            <div class="row">
                Class 'row' is used to create row that can be divided in column
                <div class="column" style="flex: 30%;">
                    Class 'column' divides the row use style="flex: 30%;" to set the width
                    <ul>
                        <li class="substep">Class 'substeb' is used to enable animation inside a slide</li>
                        <li class="substep">Second 'substep'</li>
                    </ul>
                </div>
                <div class="column">

                </div>
            </div>
            <div class="ref">
                <hr> Class 'ref' is used to insert reference
            </div>
        </div>
        <div class="slide">
            Class 'slide' enable the element to be visible in more slides
        </div>
        <div class="todo">
            Class 'todo' is used to insert some visible comment in the slide
        </div>
        <div class="notes">
            Class 'notes' is used to insert notes
        </div>
    </div>    
</div>

The object with class slide has to be out from the 'slide-body'

Create a top-down menu

Use 'data-visible-ID_SLIDE' or 'data-vin-menu-top-ID_SLIDE' to an object with class 'slide' to show the content to the slide with id=ID_SLIDE. Use To create a top-down menu use:
  • the class 'circle-top' with 'slide-name' div inside: 
                                        
<div data-vin-menu-top-down class="slide circle-top">
    <div class="slide-name">Topic 1</div>
</div>
  • the class 'circle-left' with 'slide-name' div inside: 
                                        
<div data-vin-menu-top-down class="slide circle-left">
    <div class="slide-name">Topic 2</div>
</div>
  • the class 'circle-center' with 'slide-name' div inside: 
                                        
<div data-vin-menu-top-down class="slide circle-center">
    <div class="slide-name">Topic 3</div>
</div>
